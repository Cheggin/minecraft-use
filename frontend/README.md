# Minecraft-Use Frontend

Minecraft-styled web interface for browsing and managing your schematic collection. Built with Vite, React, Tailwind CSS v4, and Convex.

## Setup

```bash
npm install
npm run dev
```

This starts both the Vite dev server and the Convex backend.

## Stack

- **Vite** + **React 19** + **TypeScript**
- **Tailwind CSS v4** for styling
- **Convex** for real-time database and file storage
- **Minecraftia** / **Minecrafter** fonts for authentic Minecraft look

## Features

- Minecraft title screen-style landing page
- Browse and search schematics from the Convex database
- Category filtering with full-text search
- Responsive layout for desktop and mobile
- All UI elements styled to match Minecraft (stone buttons, pixel fonts, beveled borders)
